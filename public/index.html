配置说明

系统配置通过环境变量或 .env 文件管理，以下为关键配置项：

WXGZH_APPID - 公众号 AppID

WXGZH_APPSECRET - 公众号 AppSecret

WXGZH_DEFAULT_AUTHOR - 默认作者

WXGZH_DEFAULT_THEME - 默认主题

WXGZH_DEFAULT_COVER_STRATEGY - 默认封面策略 (sharp / ai)

IMAGEN_API_KEY - Google Imagen 4 API Key

WEBHOOK_URL - 默认 Webhook 回调地址

API_KEY - 访问密码（设置后需登录才能访问）

修改配置后需要重启服务才能生效。
